Dataset: NIH ChestX-ray14

The NIH Chest X-Rays Dataset is a publicly available dataset with 112,120 chest x-ray images of size 1024x1024 from 30,805 different patients. Each x-ray is classified with one of the 14 disease type labels or labelled as “No Finding” for x-rays where abnormalities were not detected.

15 Classes

Atelectasis Consolidation Infiltration Pneumothorax Edema Emphysema Fibrosis Effusion Pneumonia Pleural thickening Cardiomegaly Nodule Mass Hernia No Finding

Methods

Reproducible pipeline
Preprocessing

Grayscale → Resize Images to 224×224 → Numpy → Normalization → Tensor → Multi-Label Stratified Split for Train/Validation/Test

Models

Transfer learning with ResNet‑50, DenseNet‑121, EfficientNet‑B4. Final linear head.

Data Augmentation

RandomRotation(10) and RandomHorizontalFlip()

Compiling the Model

Loss Function: Focal Loss + class weights

Optimizer: Adam

Training

Epoch loss, F1, per-label thresholds, and AUC calculated

Fine-Tuning

Unfreeze last layer of pre-trained model + modify learning rates

Evaluation

Create prediction function and evaluate on test set
