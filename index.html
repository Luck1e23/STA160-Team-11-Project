Chest X-ray Classification for Multi-Label Pathologies

We investigate deep learning approaches for automated thoracic disease detection. Using pre-trained CNNs on the NIH ChestX-ray14 dataset, we build a reproducible pipeline for curation, preprocessing, training, calibration, and error analysis.

DATASETS

NIH ChestX-ray14 processed using patient-wise splits, calibrated evaluation, and ethics-conscious test design.

Data curation

NIH ChestX-ray14

112,120 frontal chest X-rays annotated for 14 thoracic pathologies with long-tailed label distributions.

Multi-label targets Frontal view Public benchmark

Preprocessing & Splits

Images are resized to 224×224, normalized to single-channel grayscale, and split at the patient level into training, validation, and test sets.

224×224 resize Grayscale normalization Patient-wise splits

Ethics & Evaluation

Held-out evaluation prioritizes label prevalence, calibration, and sensitivity to potential shortcut features such as devices and view position.

Stratified test set Calibration checks Shortcut analysis

METHOD

A stepwise pipeline from raw DICOM/PNG files to calibrated, interpretable multi-label predictions.

Reproducible pipeline
Preprocessing

DICOM/PNG load → resize to 224×224 → grayscale normalization → optional contrast enhancement → patient-wise train/validation/test split.

Model architectures

DenseNet-121, EfficientNet-B4, and ResNet-50 initialized from ImageNet, with a final sigmoid layer for multi-label classification.

Training strategy

BCEWithLogitsLoss with class weights, AdamW optimizer, cosine learning rate schedule, and early stopping based on validation macro AUC-ROC.

Evaluation

Macro and micro AUC-ROC / AUC-PR, per-label metrics, and label-specific threshold selection using Youden’s J to balance sensitivity and specificity.

Error analysis

Grad-CAM explanations, breakdown by device and view, and inspection of representative true positives, false positives, and false negatives.

RESULTS

Model comparison and ResNet-50 performance on validation and held-out test sets.

Metrics highlight a clear improvement when moving from DenseNet-121 to ResNet-50, both in macro AUC-ROC and macro F1.

ResNet-50 summary

Macro AUC-ROC Macro F1
Validation 0.795 0.284
Test 0.757 0.295

On the held-out test set, ResNet-50 maintains macro F1 while showing a slight drop in macro AUC-ROC, consistent with a realistic generalization gap.
