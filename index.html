Datasets

NIH ChestX-ray14 and CheXpert cohorts used to benchmark multi-label thoracic pathology detection, with attention to label noise and long-tailed distributions.

Data curation

NIH ChestX-ray14

112,120 frontal CXRs annotated for 14 pathologies. Multi-label; long-tailed distributions.

Train/Valid/Test split 224×224 Augment: flip/crop

CheXpert

224,316 images with 5 competition labels (No Finding, Edema, Cardiomegaly, Consolidation, Pleural Effusion). Uncertainty handled with U-ignore/U-zero.

U-label policy CLAHE Center-crop

Test Set & Ethics

Held-out test set balanced per label prevalence. PHI-free, academic use only; models audited for shortcut learning.

Stratified Label smoothing AUC-PR focus
Method

End-to-end pipeline including preprocessing, model selection, training, evaluation, and error analysis, designed for reproducible experiments.

Reproducible pipeline
Preprocessing

DICOM/PNG read → resize 224×224 → grayscale normalization → optional CLAHE → train/valid/test split (patient-wise).

Models

Transfer learning with ResNet-50, DenseNet-121, EfficientNet-B0. Final linear head with sigmoid for multi-label BCE.

Training

BCEWithLogitsLoss + class weights; AdamW; cosine LR; early stopping on macro AUC-ROC.

Evaluation

Per-label AUC-ROC/PR, threshold tuning via Youden’s J; report macro/micro metrics; calibration with temperature scaling.

Error Analysis

Confusions by view, device, and sex; Grad-CAM sanity checks; slice-wise breakdown for long-tail classes.
