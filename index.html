Datasets

Data curation

NIH ChestX-ray14

112,120 frontal CXRs annotated for 14 pathologies.

Train/Valid/Test split 224×224 Flip/Crop

CheXpert

224,316 images with 5 competition labels.

U-label policy CLAHE Center-crop

Test Set & Ethics

Stratified, PHI-free, audited for shortcut learning.

Balanced Label smoothing AUC-PR focus

Method

Reproducible pipeline
Preprocessing

DICOM/PNG read → resize 224×224 → normalization → CLAHE → patient-wise split.

Models

DenseNet-121, ResNet-50, EfficientNet-B0 with sigmoid heads for multi-label BCE.

Training

BCEWithLogitsLoss + class weights; AdamW; cosine LR; early stopping on macro AUC-ROC.

Evaluation

AUC-ROC/PR; threshold tuning; temperature scaling for calibration.

Error Analysis

Confusions by view, device, sex; Grad-CAM; long-tail slice analysis.
