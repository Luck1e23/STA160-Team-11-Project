Chest X-ray Classification for Multi-Label Pathologies

We benchmark pre-trained CNNs to detect common thoracic findings from frontal chest X-rays. Our pipeline covers data curation, preprocessing, training, calibration, and error analysis with clear, reproducible steps.

This project explores deep learning models trained on NIH ChestX-ray14 and CheXpert, combining transfer learning and rigorous evaluation to study how well automated methods can flag clinically relevant findings from routine chest radiographs.

Datasets

Data curation

NIH ChestX-ray14

112,120 frontal CXRs annotated for 14 pathologies. Multi-label; long-tailed distributions.

Patient split 224×224 Augment: flip/crop

CheXpert

224,316 images with 5 competition labels. U-zero / U-ignore uncertainty handling.

U-policy CLAHE Center crop

Test Set & Ethics

Held-out test set balanced per label prevalence. PHI-free; audited for shortcut learning.

Stratified Label smoothing AUC-PR focus

Method

Reproducible pipeline

DICOM/PNG → normalize → resize 224×224 → optional CLAHE → patient-wise train/validation/test split.

DenseNet-121, ResNet-50, EfficientNet-B0 as backbone CNNs with sigmoid multi-label heads for 14-class classification.

BCEWithLogitsLoss with class weights, AdamW optimizer, cosine learning rate schedule, and early stopping based on macro AUC-ROC.

Per-label AUC-ROC and AUC-PR; threshold tuning via Youden’s J statistic; macro and micro metrics; temperature scaling for calibration.

Confusions stratified by projection, device type, and sex; Grad-CAM saliency to detect shortcut learning and failure modes on rare labels.
