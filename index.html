Abstract

This project explores the use of deep learning methods to automatically detect thoracic diseases from chest X-ray images. Using the NIH ChestX-ray14 dataset of 112,120 images across 14 disease categories, we implement a transfer learning approach with the XRV Pathology Classifier, a model pre-trained on large-scale medical imaging data. After preprocessing, fine-tuning, and calibration, we evaluate performance using macro/micro AUC-ROC and AUC-PR to understand both overall accuracy and class-specific behavior.

Method

We build a reproducible pipeline that goes from raw DICOM/PNG images to calibrated multi-label predictions with Grad-CAM based interpretability.

Reproducible pipeline

Preprocessing

DICOM/PNG read, resize to 224×224, grayscale normalization, optional CLAHE for contrast enhancement, and patient-wise train/validation/test split.

Model

XRV Pathology Classifier as a pre-trained backbone, with a multi-label sigmoid head for 14 pathologies, optimized with class imbalance in mind.

Training

Fine-tuning with BCEWithLogitsLoss and class weights, AdamW optimizer, cosine learning rate schedule, and early stopping on macro AUC-ROC.

Evaluation & calibration

Per-label AUC-ROC and AUC-PR, macro/micro metrics, threshold tuning (e.g., Youden’s J) and temperature scaling for better calibrated probabilities.

Error analysis & interpretability

Grad-CAM maps for true/false positives and negatives, inspection by view/device, and focused analysis of long-tail pathologies.

Results

We report validation performance with both ROC and PR metrics to capture behavior under heavy class imbalance and multi-label overlap.
