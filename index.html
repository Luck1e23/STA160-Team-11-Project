Data

Datasets

Data curation

NIH ChestX-ray14

112,120 frontal CXRs annotated for 14 pathologies. Multi-label with long-tailed class distributions.

Train/validation/test split is patient-wise to avoid leakage, with images resized to 224×224 resolution and basic augmentations applied.

Train/Valid/Test split 224×224 Augment: flip/crop

CheXpert

224,316 images with competition labels (No Finding, Edema, Cardiomegaly, Consolidation, Pleural Effusion), including uncertainty labels.

We experiment with U-ignore and U-zero policies, CLAHE contrast enhancement, and center-crop preprocessing.

U-label policy CLAHE Center-crop

Test set & ethics

Held-out test set balanced relative to label prevalence. PHI-free, academic use only; models monitored for shortcut learning.

We emphasize calibration and robustness, reporting AUC-PR alongside ROC to reflect class imbalance.

Stratified Label smoothing AUC-PR focus
Pipeline

Method

Reproducible pipeline
Preprocessing

DICOM/PNG ingest → resize to 224×224 → grayscale normalization → optional CLAHE → patient-wise train/validation/test split.

Models

Transfer learning with ResNet-50, DenseNet-121, and EfficientNet-B0. Final linear head with sigmoid outputs for multi-label BCE.

Training

BCEWithLogitsLoss with class weights; AdamW optimizer; cosine learning rate schedule; early stopping on macro AUC-ROC.

Evaluation

Per-label AUC-ROC and AUC-PR, threshold tuning via Youden’s J; macro and micro metrics; calibration with temperature scaling.

Error analysis

Confusions by view, device, and sex; Grad-CAM sanity checks; slice-wise breakdown for long-tail classes and hard negatives.
