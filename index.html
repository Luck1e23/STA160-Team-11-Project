DATASETS

NIH ChestX-ray14 and CheXpert cohorts, curated with patient-wise splits, uncertainty handling, and ethics-aware evaluation splits.

Data curation

METHOD

A reproducible pipeline for preprocessing, training, evaluation, and error analysis, designed to be dropped into any PyTorch training loop.

Reproducible pipeline
Preprocessing

DICOM / PNG read → resize to 224×224 → grayscale normalization → optional CLAHE → train/valid/test patient-wise split.

Models

Transfer learning with ResNet-50, DenseNet-121, EfficientNet-B0. Final linear head with sigmoid for multi-label BCE.

Training

BCEWithLogitsLoss with class weights; AdamW; cosine LR schedule; early stopping on macro AUC-ROC.

Evaluation

Per-label AUC-ROC / AUC-PR, threshold tuning with Youden’s J, macro and micro metrics, temperature-scaled calibration.

Error Analysis

Confusions by view, device, and sex, Grad-CAM sanity checks, slice-wise breakdown for long-tail classes, shortcut learning inspection.
