We explore deep learning methods to automatically detect thoracic diseases from chest X-ray images. Using NIH ChestX-ray14 and CheXpert, we implement a transfer learning pipeline with curated preprocessing, fine-tuning, and systematic evaluation.

Datasets

Datasets and curation

We combine large-scale public datasets with careful splitting and preprocessing to build robust training and evaluation sets for chest X-ray classification.

NIH ChestX-ray14

112,120 frontal chest X-rays annotated for 14 thoracic pathologies. Multi-label with long-tailed label distributions.

Patient-wise train/validation/test split with image resizing to 224√ó224, normalization, and standard augmentations.

Train/Valid/Test ‚Ä¢ 224√ó224 ‚Ä¢ Horizontal flip / random crop

CheXpert

224,316 images with competition labels: No Finding, Edema, Cardiomegaly, Consolidation, Pleural Effusion.

Uncertainty labels are handled with U-ignore or U-zero policies depending on the experiment.

U-label policy ‚Ä¢ CLAHE ‚Ä¢ Center-crop

Test set & ethics

Final held-out test set is balanced relative to label prevalence and kept untouched until model selection is complete.

PHI-free, academic use only; experiments checked for shortcut learning and spurious correlations.

Stratified sampling ‚Ä¢ Label smoothing ‚Ä¢ AUC-PR emphasis

Method

End-to-end pipeline

From raw DICOM/PNG files to calibrated probability outputs. Each step is designed to be reproducible and interpretable for future extensions.

Preprocessing

DICOM/PNG read ‚Üí resize to 224√ó224 ‚Üí grayscale normalization ‚Üí optional CLAHE ‚Üí patient-wise train/validation/test split.

Models

Transfer learning with ResNet-50, DenseNet-121, EfficientNet-B0 and XRV Pathology Classifier. Final linear head with sigmoid activation for multi-label BCE.

Training

BCEWithLogitsLoss with class weighting; AdamW optimizer; cosine learning-rate schedule; early stopping on macro AUC-ROC.

Evaluation

Per-label AUC-ROC and AUC-PR, threshold tuning via Youden‚Äôs J statistic, macro and micro-averaged metrics.

Error analysis

Confusions broken down by view type, device, and sex; Grad-CAM visual explanations; slice-wise analysis for long-tail pathologies.
