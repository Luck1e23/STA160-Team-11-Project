Chest X-ray Classification for Multi-Label Pathologies

We investigate deep learning approaches for automated thoracic disease detection. Using pre-trained CNNs on NIH ChestX-ray14 and CheXpert, we build a reproducible pipeline for curation, preprocessing, training, calibration, and error analysis.

METHOD

A reproducible pipeline mirroring your report: preprocessing, model selection, training, evaluation, and structured error analysis.

Reproducible pipeline
Preprocessing

DICOM/PNG load → resize to 224×224 → grayscale normalization → optional CLAHE → patient-wise train/validation/test split.

Models

DenseNet-121, EfficientNet-B4, and ResNet-50, initialized from ImageNet and adapted with a multi-label sigmoid head.

Training

BCEWithLogitsLoss with class weights, AdamW optimizer, cosine learning rate schedule, early stopping on validation macro AUC-ROC.

Evaluation

Macro and micro AUC-ROC / AUC-PR, per-label metrics, Youden’s J for threshold selection, and temperature scaling for calibration.

Error Analysis

Breakdown by pathology, view, and device, plus Grad-CAM sanity checks that highlight shortcut learning and missed findings.
