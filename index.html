Chest X-ray Classification for Multi-Label Pathologies

This project explores deep learning methods to automatically detect thoracic diseases from chest X-ray images. Using the NIH ChestX-ray14 dataset with 112,120 frontal CXRs across 14 labels, we apply transfer learning with a pre-trained X-ray pathology classifier, then fine-tune, calibrate, and analyze errors across views, devices, and demographics.

Datasets

NIH ChestX-ray14 and CheXpert are curated with patient-wise splits, uncertainty handling, and ethics-aware test selection.

Data curation

NIH ChestX-ray14

112,120 frontal CXRs annotated for 14 thoracic pathologies. We construct a patient-wise train/validation/test split to avoid leakage.

112k+ frontal CXRs 224×224 resize Flip · crop · CLAHE

CheXpert

224,316 CXRs with 5 competition labels (No Finding, Edema, Cardiomegaly, Consolidation, Pleural Effusion). Uncertainty handled via U-ignore / U-zero policies.

U-label policy Class imbalance aware View filtering

Test Set & Ethics

Held-out test set balanced by label prevalence, PHI-free and limited to academic use. We audit for shortcut learning and spurious correlations.

Stratified sampling Label smoothing AUC-PR focus
Method

A reproducible pipeline from raw DICOMs to calibrated probabilities and interpretable Grad-CAM visualizations.

Reproducible pipeline

End-to-end pipeline

1
Preprocessing

DICOM/PNG read → resize to 224×224 → grayscale normalization → optional CLAHE → patient-wise train/valid/test split.

2
Backbone models

Transfer learning with DenseNet-121, ResNet-50, and EfficientNet-B0 initialized from large-scale X-ray pre-training (XRV pathology classifier).

3
Training

BCEWithLogitsLoss with class weights; AdamW optimizer; cosine learning rate schedule; early stopping on macro AUC-ROC.

4
Evaluation & calibration

Per-label AUC-ROC and AUC-PR, threshold tuning via Youden’s J, and post-hoc temperature scaling for probability calibration.

5
Error analysis & visualization

Grad-CAM overlays, slice-wise analysis by device, view, and sex; qualitative review of true/false positives and negatives.

Training configuration

Batch size: 32 Optimizer: AdamW LR schedule: cosine decay Loss: BCEWithLogits + weights Augment: flip, crop, CLAHE Calibration: temperature scaling
